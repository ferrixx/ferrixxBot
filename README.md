## ❗️ Dependencies

```sh
- https://gradle.org | Gradle
- https://www.java.com/de | Java
- https://javacord.org | Javacord
- https://www.mysql.com | MySQL
```

## 🔌 Usage

```sh
The Command syntax is !<command> <args>
```

## ✔️ Run tests

```sh
!ping on you Discord Server to Test if the Bot is working.
```
